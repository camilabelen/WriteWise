Sebastián Rodríguez, BCSc.

Sebastián Rodríguez is a Computer Engineer from Pontificia Universidad Católica de Valparaíso.

Since 2016, he has worked in different research projects both as a Project Engineer and as a Research Assistant. The areas that involved such projects are in the domain of: Machine Learning, Natural Language Processing and Data Visualization. In regards to Software Development, Sebastian has a background in building and deploying Machine Learning based solutions with Python as main programming language. As for the domain of Web Development, his preference is using Python with HTML5 JavaScript-based libraries.

Currently Sebastian is an Associate Researcher at WriteWise doing data science and NLP in academic articles. This in order to describe, analyze and predict relevant characteristics of the articles using both Traditional Learning Machine and Deep Learning.
